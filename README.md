# FreshScan-Fruit-Quality-Assessment-App
- Fresh Scan is an Android app that performs on‑device fruit freshness detection using YOLOv8 TensorFlow Lite models - Detects and labels Apple, Guava, Mango, and Strawberry as Fresh or Rotten with real‑time camera overlays - Generates a rich analysis report: shelf‑life estimate, quality score, detected issues, nutritional facts, and storage tips.
